Accessibility testing of our page

Test text

We used ligthhouse in microsoft edge and Wave web accessibility evaluation tool to evaluate the site pages accessibility. To use these tests firstly download the extension live server. After that paste the live server link in edge and go to settings and developer tools. Then click on lighthouse and run the tests for phone and pc view. To use wave i forked the github repository and ran the repo on my own github pages. After github pages is set up i paste the link from github pages into Wave and get the results.
